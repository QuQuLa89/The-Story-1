# The-Story-1

ターミナル上で遊ぶテキストノベル形式のミニゲーム。台詞は1文字ずつタイプライター風に表示され、起動回数に応じて内容が変化する。

## 動作環境

- Python 3.10 以降
- Windows（矢印キー入力に `msvcrt`、コンソール制御に `ctypes` を使用しているため、現状 Windows 専用）
- 外部ライブラリ不要（標準ライブラリのみで動作）

## 遊び方

```bash
python main.py
```

- 台詞は自動で1文字ずつ表示され、`Enterキーで続行` の表示後に Enter を押すと次に進む
- 選択肢画面では ↑ / ↓ キーでカーソルを移動し、Enter で決定
- 一部の場面ではコマンド入力を求められる（例: `/○○`）。表示されているコマンドをそのまま入力して Enter を押す
- 2回目以降の起動では、既に選んだ選択肢の先頭に `*` が付く
- 全ての選択肢を選び終える（1〜3を選択済みにする）と隠しシナリオへ進み、クリア後は次回起動時にリプレイが表示される

## セーブデータ

`save/save.json` に起動回数・選んだ選択肢・クリアフラグを自動保存する（初回起動時に自動生成）。リセットしたい場合はこのファイルを削除する。

## ディレクトリ構成

```
main.py              エントリーポイント
game/
  display.py         文字送り表示・画面制御
  input_utils.py      矢印キーメニュー・コマンド入力の判定
  save_data.py       セーブデータの読み書き
  script.py          台詞・選択肢テキスト
  story.py           各チャプターの進行ロジック
save/                セーブデータ（Git管理外）
```
